# Wealthill.com — Static Site

A static HTML + jQuery recreation of [wealthill.com](https://www.wealthill.com), a wealth hill of books for mechanical engineers & students.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page with scrolling image banner |
| `about.html` | About the site and its author |
| `blog.html` | Blog listing with category filter |
| `gallery.html` | Image gallery with lightbox |
| `post/*.html` | Individual blog post pages |

## Blog Posts

- RESPECT vs HONOUR
- REACTING vs RESPONDING
- ARGUMENT vs DISCUSSION
- SELF-RESPECT vs EGO
- FORGIVENESS vs FORGETNESS
- Mechanical Engineering Consultancy

## Tech Stack

- Plain HTML5
- CSS3 (custom, no framework)
- jQuery 3.7.1 (CDN)
- Images served from Wix CDN

## Features

- Sticky responsive navbar with hamburger menu
- Auto-scrolling hero image banner (pauses on hover)
- Blog category filter (All / Business / Personal / Articles)
- Gallery lightbox with keyboard arrow navigation
- Contact form with animated success state
- Mobile-friendly layout

## Usage

Open `index.html` directly in any browser — no build step or server required.
